# Get-CWMSource

## SYNOPSIS
This function will list service sources on conditions.

## SYNTAX

```
Get-CWMSource [-id <Int32>] [-count] [[-condition] <String>] [[-orderBy] <Object>]
 [[-childconditions] <String>] [[-customfieldconditions] <String>] [[-page] <Int32>] [[-pageSize] <Int32>]
 [[-fields] <String[]>] [-all] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## EXAMPLES

### EXAMPLE 1
```
Get-CWMSource -Condition 'name like "*Phone*"'
```

Will return all priorities that include the word "Phone", such as "Phone Source"
